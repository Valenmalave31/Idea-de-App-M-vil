# Funcionalidades del proyecto 

## Restricciones técnicas

Esta aplicación funcionará en dispositivos Android con sistema operativo versión 6 o superior.

## Funcionalidades

1. **Entrenamiento para diferentes tipos de ejercicio:** Ofrece una variedad de opciones de ejercicio, como yoga, running, ciclismo, entrenamiento de fuerza y pilates, lo que permite a los usuarios elegir el tipo de ejercicio que deseen realizar.

2. **Planes de entrenamiento personalizados:** Permite a los usuarios seleccionar planes de entrenamiento adaptados a sus niveles de experiencia y objetivos fitness individuales.

3. **Videos demostrativos de guía:** Proporciona videos instructivos que muestran cómo realizar cada ejercicio correctamente, lo que ayuda a los usuarios a comprender la técnica adecuada.

4. **Seguimiento de progreso y estadísticas:** Permite a los usuarios realizar un seguimiento de su progreso en términos de cumplimiento de los entrenamientos, tiempo dedicado a cada ejercicio, distancia recorrida (en caso de ejercicios como running o ciclismo), calorías quemadas, entre otros.

5. **Comunidad de usuarios:** Ofrece una plataforma donde los usuarios pueden interactuar entre sí, compartir consejos, experiencias, y motivarse mutuamente para alcanzar sus objetivos fitness.

6. **Personalización de objetivos:** Permite a los usuarios establecer metas de ejercicio y seguimiento personalizado para ayudarles a alcanzar sus objetivos específicos de fitness.

7. **Recordatorios y planificación de sesiones de entrenamiento:** Ofrece la opción de establecer recordatorios para las sesiones de entrenamiento y ayuda a los usuarios a planificar su rutina de ejercicios de manera efectiva.

8. **Compatibilidad con dispositivos de seguimiento de actividad:** Integra la posibilidad de sincronizar datos de dispositivos de seguimiento de actividad física, como relojes inteligentes o bandas de fitness, para un seguimiento más preciso del progreso.
